Οδηγος χρησης: - ΕΛΛΗΝΙΚΑ

1) γράψτε στο αρχειο "items.txt" τα στοιχεια με σειρα:
<part number> <part description> <quantity>

2) τρεξτε το "write.exe" 
   (η εφαρμογη διαγραφει τα περιεχομενα του 'items.txt' για να μην υπαρχουν
    περιπτωσεις οπου ενα αντικειμενο γραφεται πολλες φορες.)

3) ανοιξτε το αρχειο '.xlsx' που δημιουργηθηκε .

4 προαιρετικο) τρεξτε το 'clear.exe' για να απαλειψετε το αρχειο '.xlsx' 

Use guide: - ENGLISH

1) in the 'items.txt' file, enter the credentials in this order:
<part number> <part description> <quantity>

2) run the "write.exe" app
  (that app deletes the contents of the 'items.txt' so there arent any cases
   in which a row duplicates itself. )

3) open the '.xlsx' file that was created. 

4 optional) run the 'clear.exe' app to clear the '.xlsx' file.

FAQ: 

'i deleted the '.xlsx' or 'items.txt' file, what do i do?'
- run the 'recreate.exe' app. That will create a blank '.xlsx' file and a blank 'items.txt' file. CAREFUL! this will clear any file that exists

'Διεγραψα το '.xlsx' η το 'items.txt' αρχειο, τι να κανω?'
- τρεξε το 'recreate.exe'. Αυτο θα φτιαξει ενα αδειο '.xlsx' και 'items.txt' αρχειο. ΠΡΟΣΟΧΗ : αυτο θα απαλειψει το αλλο αρχειο αν υπαρχει

ΣΗΜΑΝΤΙΚΟ - IMPORTANT 

ΜΗΝ ΠΕΙΡΑΞΕΤΕ ΤΟ ΑΡΧΙΚΟ '.xlsx' ΑΡΧΕΙΟ. ΚΑΝΤΕ ΜΙΑ ΑΝΤΙΓΡΑΦΗ ΣΕ ΕΝΑΝ ΑΛΛΟ ΦΑΚΕΛΟ ΠΡΩΤΑ 
DONT INTERACT WITH THE STARTING 'xlsx' FILE. CREATE A COPY FIRST IN ANOTHER FOLDER
